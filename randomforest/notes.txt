Python script make_submit.py produces Slurm submission scripts that run learn.R

Parameters
1. base directory
2. R script file
3. ontology name
4. k
5. number of features
6. output directory
7. directory for data (feature values)
8. slurm submit script filename (without extension)
9. number of training files
10. first int for training files
11. last int for training files
12. number of testing files
13. first int for testing files
14. last int for testing files
15. id for split
